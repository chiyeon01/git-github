###Linux Command###



○ pwd : 현재 경로 파악 


○ ls : 현재 디렉터리에 어떤 파일이나 디렉터리가 있는지 확인

    ● ls -a : 숨긴 파일이나 디렉터리도 표시
    ● ls -l : 파일이나 디렉터리의 상세 정보도 표시
    ● ls -r : 파일 순서를 거꾸로 표시
    ● ls -t : 파일 작성 시간순(내림차순)으로 표시


○ clear : 터미널 화면 비우기


○ cd 경로 : 현재 디렉터리 경로 이동

    ● cd .  : 현재 디렉터리로 이동
    ● cd .. : 현재 디렉터리의 상위 디렉터리로 이동
    ● cd ~  : 홈 디렉터리(사용자 디렉터리)로 이동


○ mkdir 디렉터리명 : 현재 디렉터리 안에 하위 디렉터리를 만듬


○ rm 파일명 : 파일 삭제
    ● rm -r 디렉터리명: 디렉터리 삭제


○ exit : 터미널 창 닫기
